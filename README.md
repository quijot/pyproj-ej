# pyProj-ej

Código de ejemplo de uso de [pyproj][pp].

[pyproj][pp] es una interfaz [Python][py] de la librería de proyecciones cartográficas [PROJ.4][proj] escrita en C.

# Prerequisitos

Tener instalados [Python][py], [NumPy][np] y [pyproj][pp].
